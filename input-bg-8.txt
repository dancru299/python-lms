Bài 3: Phí Giao Hàng (4 Điểm)

Đề bài: Nhập vào khoảng cách giao hàng (km) và giá trị đơn hàng (VNĐ). Tính phí giao hàng theo quy tắc sau:

Khoảng cách từ 2 km đến dưới 5 km: Phí 25.000đ.

Chữa Bài 3: Phí Giao Hàng (4 Điểm)
Đây là bài toán chia bậc kết hợp với điều kiện ưu tiên (đơn trên 300k). Cách làm tốt nhất là xét điều kiện ưu tiên trước!

Python
khoang_cach = float(input("Nhập khoảng cách (km): "))
gia_tri_don = int(input("Nhập giá trị đơn hàng (VNĐ): "))

# Xét điều kiện ưu tiên trước tiên
if gia_tri_don >= 300000:
    phi = 0
# Bắt đầu chia bậc khoảng cách nếu không được miễn phí
elif khoang_cach <= 2:
    phi = 15000
elif khoang_cach < 5:
    phi = 25000
else:
    # Lấy 25000 cộng với phần km vượt quá 5
    phi = 25000 + (khoang_cach - 5) * 5000

print("Phí giao hàng của bạn là:", phi)
Output:

Plaintext
Nhập khoảng cách (km): 7.5
Nhập giá trị đơn hàng (VNĐ): 150000
Phí giao hàng của bạn là: 37500.0
Giải thích từng dòng:

Dòng 1-2: Nhập dữ liệu, ép kiểu float cho khoảng cách và int cho giá trị đơn.

Dòng 5: Lệnh if kiểm tra ngay xem đơn có lớn hơn 300k không. Nếu đúng, gắn phí = 0 và BỎ QUA toàn bộ tính toán bên dưới.

Dòng 8-11: Nếu đơn < 300k, máy tính mới kiểm tra tiếp các bậc khoảng cách.

Dòng 13: Công thức tính bậc cuối. Lấy tổng km trừ đi 5 để ra số km vượt mức, rồi nhân với 5000.

Bài 2: Tính chu vi hoặc diện tích (Độ khó: Trung bình)

Mục tiêu: Rẽ nhánh dựa trên chuỗi nhập vào.

Python

if lua_chon == "Chu vi":
    print("Kết quả:", canh * 4)
elif lua_chon == "Diện tích":
    print("Kết quả:", canh * canh)
else:
    print("Lựa chọn không hợp lệ")

Bài 4: Tính thuế thu nhập (Độ khó: Khó)

Output: 400000.0

Đáp án mẫu:

Python
luong = float(input("Nhập lương (VNĐ): "))

    thue = 0
else:
    thue = (luong - 11000000) * 10 / 100
print("Thuế phải nộp là:", thue)
